# 🚀 EliteTrades — Trade Smarter. Profit Bigger.

EliteTrades is a state-of-the-art, premium trading platform landing page designed with a high-end **Glassmorphism** aesthetic. It provides a seamless experience for traders looking for premium signals, advanced market analysis, and educational resources.

---

## ✨ Features

- **💎 Elite Design**: Modern glassmorphism UI with vibrant neon accents and smooth micro-animations.
- **📊 Interactive Signal Preview**: Animated candlestick charts and live-badge status indicators.
- **⚡ Performance First**: Built with Vite for ultra-fast loading speeds and optimized asset delivery.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.
- **🛡️ Secure Checkout**: Integrated pricing tiers with multiple payment options (Credit Card, Crypto, and Direct Messaging).
- **📉 Real-time Stats**: Dynamic statistics bar showing win rates, active users, and signal frequency.

---

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Modern CSS3 (Variables, Flexbox, Grid, Glassmorphism)
- **Tooling**: [Vite](https://vitejs.dev/)
- **Animations**: Custom CSS Animations & Canvas Particles
- **Icons**: [Lucide React](https://lucide.dev/) (referenced in components)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sathya1245632/Trading-website-.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Trading-website-
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
